# 海之心通用预约系统


[查看演示](http://ty.ohyu.cn/)

### 为什么开源
到目前为止我们开发了很多系统，也有一部分系统开发出来后就闲置了，与其这样不如把低版本代码开源出来，这样既能满足小需求或者实力比较小的公司低成本使用，也能让更高需求的客户找到我们。

### 介绍
通用预约系统是成都海之心科技有限公司开发的一款适用性比较广的一款系统。目前开源的版本【预约项目+预约时间+表单+核销】适用觉大多数简单场景的预约。但是本套系统目前的实际的功能更多适用的范围更广。

之前这套系统做过很多行业的预约，学校，博物馆，图书馆等等。

可做会议预约、驾校预约、医疗预约、活动预约、培训预约等多场景的预约需求。

### 安装部署

本前端使用uniapp开发，适配了微信公众号、微信小程序、手机H5网页 三端

前端配置。
* 1、修改main.js里面的域名，main.js里面的appid是公众号网页
* 2、修改common/vmeitime-http/interface.js里面的域名
* 3、小程序的分享在static/js/fenxiang.js里面

后端也开源了：[开源后端](https://github.com/ohyueo/general)
